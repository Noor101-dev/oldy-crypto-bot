# OLDY CRYPTO Telegram Bot 🤖

Auto-runs every 5 minutes via GitHub Actions.

## Commands
| Command | Description |
|---|---|
| `/start` | Bot intro |
| `/meme` | Random Hindi/English crypto meme |
| `/quiz` | Crypto knowledge quiz poll |
| `/profile` | Your info + rank |
| `/price` | Live crypto prices |
| `/leaderboard` | Most active members |

## Setup

### Add Bot Token Secret
- Go to repo → **Settings** → **Secrets and variables** → **Actions**
- Click **New repository secret**
- Name: `BOT_TOKEN`
- Value: your bot token from @BotFather

### Enable Actions
- Click the **Actions** tab → Enable workflows
- Run manually: Actions → **Telegram Bot** → **Run workflow**

Bot runs automatically every **5 minutes** 🚀
